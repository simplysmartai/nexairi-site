# Pre‑Publish Checklists

Content
- [ ] Clear lede + nut graph within 150 words
- [ ] Each section solves one problem; subheads every ~150 words
- [ ] 2–4 internal links (see /data/internal_links.json)
- [ ] 2–4 external citations (credible; no content farms)

Facts & Safety
- [ ] Numbers and temps have sources
- [ ] Safety guidance is conservative and accurate
- [ ] Disclaimers where needed (health, travel, legal)

SEO & Meta
- [ ] Title ≤ 70 chars, includes target keyword
- [ ] Meta description 150–160 chars
- [ ] Keywords present in H2/body; no stuffing
- [ ] Canonical set; OG/Twitter tags filled

Accessibility
- [ ] Alt text meaningful
- [ ] Bullets and short paragraphs
- [ ] Color contrast and plain language

Images
- [ ] image_prompt written for hero and section figures
- [ ] Thumbnails have clear, uncluttered subjects
